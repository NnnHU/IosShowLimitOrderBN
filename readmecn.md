# 项目说明

本项目是一个苹果 iPhone 应用程序，旨在实时获取市场深度数据，进行买卖比率和大单监控，并生成文本和图表输出，最终将结果推送到指定渠道。

## 项目背景

- **应用类型**: 苹果 iPhone APP
- **技术参考**: 本项目参考 `otherCODE` 目录下的 Python 项目。
- **界面设计**: APP 的界面风格应参考上述 Python 项目发送到 Discord 的消息样式。

## 应用架构设计

### 核心模块
- **数据获取模块**: 实时获取市场深度数据
- **分析模块**: 计算买卖比率和大单监控
- **输出模块**: 生成文本和图表输出
- **推送模块**: 将结果发送到指定渠道

### 技术栈
- **前端**: SwiftUI
- **数据层**: CoreData + Alamofire
- **分析层**: 自定义算法
- **推送层**: Discord API 集成
